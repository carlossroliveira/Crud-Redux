## Crud - redux

CRUD (Create, Read, Update, and Delete) are the four basic operations (data creation, query, update, and destruction).


```bash
# Access the project folder and download dependencies.
$ yarn install
or
$ npm install
```

```bash
# Run
$ yarn start
or
$ npm start
```
